About Accra Weather

Delivering accurate, localized weather data for Accra, Ghana — accessible, responsive, and user-friendly.

Project Purpose

The Accra Weather app was developed to make reliable and localized weather forecasts accessible to everyone in Accra. It is designed with performance, accessibility, and responsiveness in mind, while showcasing API integration and modern front-end techniques.

Key Features

Real-time Data

Pulls live weather updates from OpenWeatherMap API, with fallback data for offline testing.

Responsive Design

Optimized for all devices with mobile-first design and accessible navigation controls.

Unit Preferences

Save and switch between Celsius and Fahrenheit easily, with settings stored locally.

Accessibility

Semantic HTML, ARIA attributes, and keyboard-friendly interactions for all users.

Technology Used

  • HTML5 — semantic and accessible markup
  • CSS3 — responsive, mobile-first layout
  • JavaScript (ES6 modules) — modular architecture
  • OpenWeatherMap API — real-time weather data
  • Local JSON — offline / fallback data support
